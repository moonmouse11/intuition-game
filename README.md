# Интуиция — Игровое табло

Интерактивное веб-табло для проведения игры «Интуиция» в стиле одноимённого телешоу. Ведущий управляет табло с клавиатуры, проецируя экран на большой дисплей или стену.

## Как это работает

На экране отображается список из 9 утверждений, расположенный в левой части экрана. Перед проекцией стоят участники (актёры). Игрок выбирает утверждение и сопоставляет его с одним из участников. Ведущий с клавиатуры управляет табло: выделяет строку, подтверждает или отклоняет ответ.

При выборе варианта список скрывается и выбранная строка показывается по центру экрана. После звукового сигнала:
- **Верный ответ** — строка 1.5 секунды светится зелёным, затем исчезает
- **Неверный ответ** — строка 1.5 секунды трясётся, затем возвращается в список

Когда все варианты угаданы, на экране запускается салют и появляется заставка «Интуиция».

## Управление

| Клавиша | Действие |
|---------|----------|
| `1` – `9` | Выбрать вариант — список скрывается, выбранная строка центрируется |
| `Y` | Верный ответ — зелёное свечение, затем строка исчезает |
| `N` | Неверный ответ — анимация тряски, затем строка возвращается в список |
| `Space` | Переместить список в центр / обратно влево |
| `R` | Новая игра (сброс всех вариантов) |
| `Esc` | Пауза — показать заставку «Интуиция» без сброса игры |
| `H` | Показать / скрыть подсказки управления |

Также можно кликать мышью по строкам для выбора.

## Звуковые эффекты

- **select.mp3** — воспроизводится при выборе варианта
- **positive.mp3** — воспроизводится при верном ответе (`Y`)
- **negative.mp3** — воспроизводится при неверном ответе (`N`)

## Запуск

Открыть `index.html` в браузере. Для корректной работы звуков рекомендуется запускать через локальный сервер:

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```

Затем открыть `http://localhost:8000` в браузере.

## Настройка вариантов

Список утверждений задаётся в массиве `items` в файле `app.js`:

```js
const items = [
  "Собирался открыть чебуречную",
  "Цветочник(ца)",
  // ...
];
```

Замените строки на свои — табло автоматически подстроится под любое количество вариантов.

## Рекомендации по использованию

- Разворачивайте браузер на полный экран (`F11`) для максимального эффекта
- По умолчанию список расположен слева — актёры могут стоять в правой части перед проекцией
- Нажмите `Space`, чтобы переместить список в центр (например, для демонстрации без участников)
- Нажмите `H` перед началом, чтобы показать зрителям подсказки по управлению, затем скройте их
- Нажмите `Esc` для паузы между раундами без потери прогресса
