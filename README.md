# Интуиция — Игровое табло

Интерактивное веб-табло для проведения игры «Интуиция» в стиле одноимённого телешоу. Ведущий управляет табло с клавиатуры, проецируя экран на большой дисплей или стену.

## Как это работает

На экране отображается список из 9 утверждений. Перед экраном стоят участники (актёры). Игрок выбирает утверждение и сопоставляет его с одним из участников. Ведущий с клавиатуры управляет табло: выделяет строку, подтверждает или отклоняет ответ.

При сдвиге таблицы влево (пробел) освобождается место — актёры перед проекцией не перекрывают список. Справа появляется неоновая вывеска с названием выбранного варианта.

## Управление

| Клавиша | Действие |
|---------|----------|
| `1` – `9` | Выбрать вариант из списка (подсвечивается красным) |
| `Y` | Угадал — строка исчезает с зелёной вспышкой |
| `N` | Не угадал — выделение снимается, строка возвращается |
| `Space` | Сдвинуть таблицу влево / вернуть в центр |
| `R` | Сбросить игру (все строки возвращаются) |
| `H` | Показать / скрыть подсказки управления |

Также можно кликать мышью по строкам для выбора.

## Звуковые эффекты

- **select.mp3** — воспроизводится при выборе варианта
- **positive.mp3** — воспроизводится при верном ответе (`Y`)
- **negative.mp3** — воспроизводится при неверном ответе (`N`)

## Запуск

Открыть `index.html` в браузере. Для корректной работы звуков рекомендуется запускать через локальный сервер:

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```

Затем открыть `http://localhost:8000` в браузере.

## Настройка вопросов

Список вариантов задаётся в массиве `items` в файле `app.js`:

```js
const items = [
  "Собирался открыть чебуречную",
  "Цветочник(ца)",
  // ...
];
```

Замените строки на свои — табло автоматически подстроится.

## Рекомендации по использованию

- Разворачивайте браузер на полный экран (`F11`) для максимального эффекта
- Используйте `Space` для сдвига таблицы, когда актёры стоят перед проекцией
- Нажмите `H` перед началом, чтобы показать зрителям подсказки, затем скройте их
